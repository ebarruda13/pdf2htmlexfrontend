pdf2htmlEX Frontend Manual de Utilizao

Este programa destina-se a converso em massa para o formato HTML, a partir de arquivos originalmente no formato PDF, para atender demanda do portal de transparncia.

O instalador do programa pode ser baixado a partir do link a seguir.

https://tinyurl.com/instala-pdf2htmlexfrontend-1x

A instalao simples e rpida, bastando seguir os passos informados. Aps concluir a instalao, o cone abaixo surgir na rea de trabalho:

Quando aberto, esta janela aparece:

O boto Selecionar arquivos PDF faz a seleo dos arquivos de origem:

possvel selecionar mais de um arquivo por vez.

Tambm possvel arrastar e soltar arquivos do explorador do Windows para a janela do programa.

O boto Remover da lista limpa uma seleo especfica de arquivos selecionados para converso. O boto Remover todos da lista deixa a lista totalmente em branco.

Clique duplo em algum arquivo na lista acima permite abri-lo para visualizao.

O campo abaixo permite incluir cdigo extra no arquivo HTML. O cdigo extra fica armazenado no arquivo XML de configurao do programa, acessvel com clique duplo em seu nome na aba Manual.

Aps selecionar os arquivos, clique em Converter para iniciar a converso. Este processo pode demorar dependendo do desempenho do seu sistema e do tamanho dos arquivos de origem. A janela do programa pode congelar momentaneamente.

Ao fim, o programa informa a concluso e os arquivos so gerados no mesmo caminho dos arquivos originais ou numa pasta especfica caso tenha sido selecionada pelo boto Selecionar pasta de sada.

Caso queira abrir rapidamente o arquivo HTML convertido, basta dar um clique duplo no PDF correspondente que aparece na lista de arquivos do programa.
